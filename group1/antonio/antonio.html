Computer Languages: Java

Object Oriented Programming Language

About Java

Java is a high-level, class-based, object-oriented programming language. Syntax is similar to C and C++. Java is compiled with bytecode. This design helps java become more portable becauset programs written for the Java platform will run similarly on any combination of hardware and operating system with adequate run time support.
