JavaScript: Bringing Web Pages to Life!

What Is JavaScript?

The Oxford Languages Dictionary defines it as "an object-oriented computer programming language commonly used to create interactive effects within web browsers." JavaScript is what is known as an lightweight interpreted programming language. This means that it is read by a different program called an interpreter at the time of execution instead of being compiled before running. Because of this, an interpretive language can make changes instantly to the output and is interactive.

So, What Does JavaScript Do?

If you have ever ordered online then you have probably seen JavaScript in action! While HTML and CSS are used for the style and structure of a website, JavaScript (often called JS) adds functionality through interactive elements. Essentially, it is "a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive." (www.hackreactor.com) An example is clicking on a pair of pants and adding it to your cart while shopping. A JS function takes the information and adds it to the "code", moving the pants from their inventory to your cart for checkout.
